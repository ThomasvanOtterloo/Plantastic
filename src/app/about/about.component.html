Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

Plantastic

Het idee achter deze applicatie is dat iedereen zijn eigen planten kan verkopen en kopen. Gebruikers kunnen elkaars planten bekijken en een review plaatsen om de planten die ze gekocht hebben. Als gebruiker kan je andere gebruikers volgen om hun favoriete planten te bekijken. Als ik meer tijd had wilde ik graag een community form maken waarbij de gebruikers elkaar kunnen helpen met hun planten met gebruik van posts en comments.

Entiteiten

Hier geef je een toelichting op je entiteiten. Voeg ook een afbeelding van je model toe zoals dat bij NoSql gevraagd wordt. Door Plantastic functioneel te maken voor gebruikers zijn er een aantal entiteiten die ik nodig heb. De entiteiten die ik heb gebruikt zijn User, Product, Order, Review en Category.

Product

  • id: string
  • name: string
  • description: string
  • price: number
  • image: string
  • Category: Category

Review

  • Author: User
  • Description: string
  • Rating: number
  • Date: Date

Order

  • Total: number
  • Delivery: Date
  • Quantity: number

Userstories

Hier verwoord je in de vorm van userstories de functionaliteit van jouw applicatie.

Nr Userstory Evt toelichting
1 Als gebruiker wil ik een overzicht van planten kunnen raadplegen, zodat ik kan zien welke planten op dit moment verkocht worden door andere gebruikers van de website Overzicht is een grid.
2 Als ingelogde gebruiker wil ik zelf een plant kunnen verkopen, zodat ik mijn eigen planten in het systeem kan invoeren en verkopen.
3 Als ingelogde gebruiker wil ik mijn eigen planten kunnen bewerken en verwijderen, zodat ik mijn informatie up-to-date kan houden.
4

Als ingelogde gebruiker wil ik een review kunnen plaatsen op verkopers, zodat ik mijn mening kan uiten op een product die verkocht wordt op deze website.
5

Als ingelogde gebruiker wil ik mijn review kunnen editen of verwijderen, zodat ik eventueel mijn mening kan veranderen met het gene dat ik opgeschreven heb als review.
6

Als gebruiker wil ik een plant kunnen kopen, zodat ik een plant kan kopen van een andere gebruiker.

7

Als gebruiker wil ik een account kunnen aanmaken op de website, zodat ik gebruik kan maken van de website en mijn eigen planten kan verkopen of kopen.

8

Als gebruiker wil ik met een account kunnen inloggen op de website, zodat ik gebruik kan maken van de website.
